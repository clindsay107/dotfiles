Helpful commands or snippets that would be nice to copy/paste

## IRC

List tmux sessions:

    $ tls

Attach to existing named tmux irc session locally:

    $ tmux a -d -t irc  

Create new tmux irc session locally:

    $ tmux new -s irc 'irssi'
